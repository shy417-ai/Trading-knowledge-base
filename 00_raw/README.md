# 00 Raw Layer

统一存放原始层文件：未经处理的资料、临时想法、语雀迁移文档、盘中观察、LLM 对话摘要，以及从材料中初步提炼出的 claim 卡片。

这里的内容还没有进入稳定知识库。进入 `03_concepts/`、`04_playbooks/`、`05_system/` 之前，都先保留在这里。

建议命名：

- `YYYY-MM-DD_source_topic.md`
- `YYYY-MM-DD_book_chapter.md`
- `YYYY-MM-DD_video_notes.md`
- `YYYY-MM-DD_market-note.md`
- `YYYY-MM-DD_claim_topic.md`

每份资料至少记录：

- 来源链接或书名
- 作者
- 日期
- 市场/品种/时间框架
- 你为什么收集它

## Content Types

- Raw source: PDF、网页摘录、书摘、课程笔记、语雀导出文档。
- Inbox note: 临时假设、盘中观察、待研究问题、交易灵感。
- Extract note: 从资料或交易记录中初步提炼出的 claim、机制、风险、反例。

## Weekly Cleanup

- 稳定概念移到 `03_concepts/`。
- 可执行交易剧本移到 `04_playbooks/`。
- 系统级规则或检查清单移到 `05_system/`。
- 暂时无法验证的内容继续留在这里，并标记 `status: parked`。

不要在这里追求完美整理。这个目录的价值是保留原始证据和待加工材料。
