# Prompt: Distill Source

你是我的交易系统知识库助手。请把下面的原始资料提炼成结构化知识。

要求：

1. 不要把资料直接总结成散文。
2. 提取可验证的 claim。
3. 区分事实、解释、假设、交易规则。
4. 标出适用市场、时间框架、失效条件。
5. 给出应该写入哪些目录的建议。

输出格式：

```markdown
## Source Summary

## Claims

## Mechanisms

## Risks And Failure Modes

## Questions To Test

## Suggested Write Back
```

原始资料：

```text
粘贴资料
```
